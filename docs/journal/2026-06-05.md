## Did

### Firewall rules (pfSense → Firewall → Rules)

Wrote zone isolation rules across all four interfaces:

**VULNERABLE**
- Block VULNERABLE subnets → 10.10.0.0/16 (all internal lab zones)
- Pass VULNERABLE subnets → any (internet access)

**LAN (SERVICES)**
- Block LAN subnets → 10.10.20.0/24 (TRUSTEDMGMT)
- Block LAN subnets → 10.10.40.0/24 (MONITORING)
- Default allow LAN to any (internet) — pre-existing, left in place

**TRUSTEDMGMT**
- Pass TRUSTEDMGMT subnets → any (full admin access)

**MONITORING**
- Pass any → MONITORING subnets (allow Wazuh agent inbound)
- Pass MONITORING subnets → any (allow internet for updates)

Applied all changes. Default-deny posture between zones is now live.

### LXC migration into VLAN 10

Migrated all 6 LXC containers from the flat home network (192.168.0.x / vmbr0) to
the SERVICES zone (10.10.10.x / vmbr1). Used `pct set` commands from the Proxmox host
shell for speed and reliability.

**Key lesson learned:** LXC containers connect to `vmbr1` with **no VLAN tag** — pfSense
handles VLAN segmentation internally via its sub-interfaces. Setting `tag=10` on the
container caused "Destination Host Unreachable" because pfSense's untagged LAN interface
(`vtnet1`) doesn't accept tagged frames.

## Broke / fixed

- **VLAN tag issue:** Setting `tag=10` on LXC net0 caused no connectivity. Fixed by
  removing the tag (`pct set <CTID> -net0 ... type=veth` without `tag=`). pfSense sees
  all vmbr1 traffic as untagged on its LAN interface and handles VLAN routing internally.

- **LXC console unresponsive:** CT 104 console froze after migration. Worked around by
  using `pct exec 104 -- <command>` from the Proxmox host shell instead.

## Learned

- pfSense LAN = untagged trunk port on vmbr1; containers should connect untagged.
- `pct set` is faster and more reliable than editing network config in the Proxmox GUI.
- Default-deny firewall rules must be placed **above** allow-all rules (first match wins).
- The 10.10.0.0/16 supernet covers all four VLANs in a single block rule — cleaner than
  four separate rules.
